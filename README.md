# finpulse-ml

Модель для прогнозирования персональных расходов на следующую неделю на основе исторических данных. Использует робастные статистические методы для работы с волатильными данными.

Возможности:
- Прогнозирование трат на следующую неделю
- Доверительные интервалы для безопасного планирования
- Анализ паттернов расходов и сезонности
- Персональные рекомендации по бюджету
- Визуализация трат по категориям
- Обнаружение аномалий и выбросов
